# Fluoride Adsorption Hybrid Model

Project scaffold for hybrid (Langmuir + ML) modeling of fluoride adsorption on coconut husk.

Folder layout:
- data/: DoE and dataset CSV files
- src/: scripts (doe_design.py, simulate_adsorption.py, etc.)
- notebooks/: exploratory notebooks
- models/: saved model artifacts
- app/: Streamlit dashboard
- reports/: generated reports
- output/: figures and images

See NEXT_STEPS.md for execution plan.
