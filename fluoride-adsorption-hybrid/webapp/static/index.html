lightbulb Key Takeaways

  • The 24-parameter model meets the project's ≥0.94 R² target using all 10 water-chemistry factors + 14 engineered features.
  • The 4-parameter model trades some accuracy for practicality — using only pH, Dose, Initial F⁻ Concentration, and Contact Time, the variables cheapest to measure/control in the field.
  • On the same simulated dataset (controlled ablation), dropping the 6 secondary factors costs about 0.05 R² — because those factors are statistically independent (LHS design) of the 4 kept ones.
  • The current 4-parameter model is validated on real experimental data (15% of its training set), giving it direct lab credibility the 24-parameter model (fully simulated) does not have.
